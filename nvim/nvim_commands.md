  Basic Commands

Save only: Type :w filename and press Enter. 
	For example, :w my_new_file.txt.

Save and Quit: Type :wq filename and press Enter.

Quick Save & Quit: Type :x filename and press Enter 
	(only saves if changes were made).


Temporary Row Numbers (Current Session)
To enable row numbers for your current session, 
	press Esc to enter Normal Mode 
	and type one of the following commands: 

Absolute Numbers: Type :set number (or :set nu) and press Enter.

Relative Numbers: Type :set relativenumber (or :set rnu). 
	This shows how many lines away other rows are from your current cursor position.

Hybrid Numbers: Type :set number relativenumber to see the absolute number for the current line and relative numbers for all others.
Disable Numbers: Type :set nonumber or :set norelativenumber to hide them.
